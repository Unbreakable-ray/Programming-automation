# git commit message.

Small AHK script to write git commit message faster with just 1-2 keys for one word.

<br><br><br>

## How to use it:

- Open GitHub Desktop.

- Run the script (it will only works for `GitHubDesktop.exe`).

- Press `F1` to know the key combinations / Shortcuts.
